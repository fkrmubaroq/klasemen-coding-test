## Getting Started

First, run app :

```bash
# step 1
npm install 

# step 2
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```


Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
NodeJS Version : V18.17.0
